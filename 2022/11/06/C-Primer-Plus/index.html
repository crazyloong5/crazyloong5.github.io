C语言概述

C程序是什么样子的？

先来看一个简单的C程序，看看之间是否能目标该程序的用途，之后会做出详细的解读。

解读

  1. #include <stdio.h> <= 包含另一个文件

#include是C预处理指令。<stdio.h>是所有C编译器软件包都提供的头文件，它由C开发者所写，内部包含了一些常量、函数。你可以这么去理解这一行代码：#include 拷贝了stdio.h中的代码，并用这些代码替换掉自己，到此预处理完毕，程序从头开始编译。

stdio.h提供键盘输入和屏幕输出的支持。也就是让你可以输入和输出

  1. int main(void) <= 函数名

C程序包含一个或多个函数，他们是C程序的基本模块。上方程序中的main()就是一个函数。圆括号表示main()是一个函数名。int表明main()函数返回一个整数(先不细讲)，void表明main()不含任何参数。

C程序一定是从main()函数开始执行。除了main()函数，你可以任意命名其他函数，但main()函数一定是开始的函数。

  1. /* 一个简单的C程序 */ <= 注释

​ // 这种注释只能写一行

注释在/*和*/之间，用于提高程序的可读性，不参与编译。

  1. { <= 函数体开始

  2. int num; <= 声明

该声明表明，将使用一个名为num的变量，并且num是int(整数)类型。编译器使用这些信息为num变量在内存中分配存储空间。

int是C语言中的一个关键字，表示C的一种数据类型。不能用int作为函数名和变量名。

num是你所命名的一个标识符，也就是一个变量、函数或其他实体的名称。

声明把特定标识符与计算机内存中的特定位置联系起来，同时也确定了存储在某位置的信息类型或数据类型。

在C语言中，所有变量必须先声明才能使用。

命名规则：可以使用大小写字母、数字和下划线来命名。名称的第一个字符必须是字符或下划线，不能是数字。C语言的名称区分大小写。

有效名称 无效名称
wiggles $2]**
cat2 2cat
Hot_Tub Hot-Tub
taxRate tax rate
_kcab don’t
  1. num = 1; <= 赋值表达式语句

在声明中我们给了num一个空间，在这里我们把1这个值存到这个空间里。赋给num的值可以不同，所以我们也可以叫num为变量。（赋值表达式从右边把值赋到左边）

  1. printf("computer.\n"); <= 调用一个函数

该语句调用printf()函数，在屏幕上显示I am a simple。\n表示换行，对于printf来言，他的意思是”在下一行的最左边开始新的一行”。

在程序中使用函数叫做调用函数。

printf为函数名，这个函数会看括号内双引号中的内容，并将其打印到屏幕上。

  1. printf("My favorite number is %d because it is first.\n",num);

调用printf()把num的值(1)放置在用双引号括起来的内容中一起打印。%d告诉计算机以何种形式输出num的值、打印在何处。

  1. return 0; <= return语句

可以暂时把该行看作是结束main()函数的要求。如果省略main()函数末尾的return语句，程序在运行至最外面的花括号是会返回0。

  1. } <= 函数体结束

一般而言，所有C函数都使用花括号标记函数体的开始和结束，这是规定。

花括号还可以用于把函数中的多条语句合并为一个单元或块。
