注解开发

我们也可以使用注解的形式来进行开发，用注解来替换掉xml

使用注解来映射简单语句会使代码显得更加简洁，但对于稍微复杂一点的语句，Java 注解不仅力不从心，还会让你本就复杂的 SQL 语句更加混乱不堪，所以我们在实际企业开发中一般都是使用XML的形式

​
